How to install Symmetrical OS:

Step 1: Install an ISO image writing program

We recommend installing a software such as Balena Etcher to create your installer disk.

A few types of media you can use for this include:

  • DVDs
  • USB stick / Thumbdrive
  • SD card
  • Separate partition

These steps will not work with:

  • CDs (Not enough space on a disk)
  • Your pet dog (not enough energy to process data)
